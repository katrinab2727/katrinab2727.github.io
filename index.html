Spotify Charts Song Picker App

This project showcases a game I developed using Code.org's App Lab. It uses data directly from Spotify's Top 50 USA, Viral 50 USA, Top 50 Worldwide, and Viral 50 Worldwide charts. It updates everyday and the app randomly selects a song from the list chosen. It also states its rank on the list and how many of the 4 lists it is on. The game demonstrates skills in:

  • JavaScript programming
  • Game design
  • User interface (UI) development
